# 散列容器

#### 简介
自定义结构体可以通过重载比较函数实现，但更建议使用散列函数模板特例。同时对于自定义类型，需要重载“==”或者申明std::equal_to特列，因为散列表容器模板还要求equal_to<key>用于比较key真实内容是否相等
```C++
#include <functional>
struct my_data
{
  int id;
  std::string name;
  bool operator== (const my_data &a)
  {
    return id == a.id && name == a.name;
  }
}
namespace std
{
  template<>
  struct hash<my_data>
  {
    hash<int> ifunc;
    hash<std::string> sfunc;
    size_t operator() (my_data const &a) const
    {
      return ifunc(a.id) ^ sfunc(a.name);
    }
  }
  
  template<>
  struct equal_to<my_data>
  {
    bool operator()(my_data const &a, my_data const &b) const
    {
      return a.id==b.id && a.name==b.name;
    }
  }
}
```
